You can download the teaching activities from Compass in CSV format. You need to upload them with the following columns:

Reference ID
The ID number of the activity as stored in Compass
Name
The name of the activity
Week
The week during which the activity runs, or the category to which the activity belongs. This will change depending on whether the block is structured into weeks or not.
Position
Specifies the ordering of activities in the week. Called sequence number in Compass.
Activity Type
The type of activity. Currently accepts {{ accepted_types|humanize_list }}

The first row of the file should be the column headings, spelt the same as above. It doesn't matter what order the columns are in, and any extra columns are ignored.

This form currently only accepts CSV files.

Please note that the weeks/categories for the activities will be sorted by the order in which they appear in the CSV spreadsheet.
