Question Approval

How do I select the questions I want to approve?

Click on the "Approve" tab in the navigation bar at the top of the page to get to the Approval Dashboard.. Then choose a block from the list and choose the teaching activity that you want to approve.

How do I actually approve the questions?

Click on the "Approve" table in the navigation bar at the top of the page to go to the Approval Dashboard. Scroll down and press the button "Approve questions".

Which questions do I approve and which do I edit and which do I flag?

Edit the question if there is a spelling error that is easy to rectify.

Flag a question if

  1. It has more than one correct answer
  2. All the answers are incorrect
  3. The explanation is unclear

Approve a question if there is no need to edit or flag it.

If a question is really good then it is probably suitable for a model quiz, or to be sent to the Faculty. If this is the case, click "Edit Attributes", check the box and click "Save".

I don't want to do this! It will take me ages!

No, approving should not take ages. Don't spend more than 5 minutes on one question. If you are unsure, flag the question. You don't need to rewrite the question. The question will be sent back to the person who wrote it, and he/she will rewrite it.
